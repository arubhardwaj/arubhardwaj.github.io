I usually code in R, Python and Julia. On this webpage, there are few codes written in R and some codes can be found on my Github profile

[**Application of Structural Vector Autoregression (SVAR) Model in R**](/codes/PhilippinesBQ.html) <br/>
Using the vars package in R, I produced codes for studying SVAR from the framework developed by Blanchard and Quah (1989). Replicated codes of that paper by Oliver Blanchard and Danny Quah are availiable [**here**](/codes/Replication BQ.html)

[**Interactive Time-Series Plot of US Unemployment Rates (1948-2020)**](/codes/USunrate.html) <br/>
Time series plot of unemployment rates of the Unites States, made in R using ``plotly``. Data used here is from the U.S. Bureau of Labor Statistics.

[**Development Index of SAARC Countries**](/codes/saarcindex.html) <br/>
In this project, I developed an index, on the basis of selected indicators, of SAARC countries. The data used in this project is of 2018.

[**Animated Plot using gganimate**](https://rpubs.com/arubhardwaj/gdp-and-per-capita) <br/>
gganimate is a package in R which allows you to create dynamic plots. The code above use data from Gapminder.org, imported directly from the site, thourgh the package called gapminder. The plot shows the relationship between GDP percapita and life expectancy in the time series
