# [WooldridgeDatasets.jl](https://github.com/arubhardwaj/WooldridgeDatasets.jl)
WooldridgeDatasets.jl is a registered package, in ``Julia``, which allows to download the datasets from ``Jeffrey Wooldridge's Introductory Econometrics`` textbook. For using the package, run ``Pkg.add("WooldridgeDatasets")`` and then ``using WooldridgeDatasets``. After that, obtain the data using ``wooldridge("dataset name")``, for example: ``wooldridge("wage1")``. Names of all datasets are same as in book.
